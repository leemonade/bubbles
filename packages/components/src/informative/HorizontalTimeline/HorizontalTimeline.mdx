## Overview

HorizontalTimeline is a component to render a timeline of events distributed based on the time between them.

<ThemeProvider>
  <HorizontalTimeline
    rootStyles={{ padding: 24 }}
    locale="es-ES"
    color="negative"
    data={[
      { label: 'Visualization', date: new Date('2022-05-01') },
      { label: 'Start', date: new Date('2022-05-09') },
      { label: 'Deadline', date: '2022-06-10T00:00:00.000Z' },
      { label: 'Evaluation', date: new Date('2022-06-11') },
      { label: 'Results', date: new Date('2022-06-15') },
    ]}
  />
</ThemeProvider>

The `data` prop must be an array of objects with this shape:

```js
{
  label: 'Event label',
  date: Date
}
```
