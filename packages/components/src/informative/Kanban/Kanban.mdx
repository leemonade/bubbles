## Overview

Kanban is a component that displays a kanban board. You must provide a `value` which consists of a columns array property,
the items of the this array will have this shape:

```js
{
  id: '734c8a98-ce1b-4d8a-912f-f65eeb850f24',
  title: 'Column title',
  cards: [
    {
      id: 'a5182556-be4e-452f-ad42-36c3490d2771',
      title: 'Task title',
      calendar: '9c1ec044-beb7-4f86-b1db-0c8b365ee786',
      startDate: '2022-01-05T09:03:10.000Z',
      endDate: '2022-01-06T09:03:10.000Z',
      isAllDay: 0,
      repeat: 'dont_repeat',
      type: 'plugins.calendar.task',
      status: 'active',
      data: { column: '734c8a98-ce1b-4d8a-912f-f65eeb850f24' },
      deleted: 0,
      created_at: '2022-01-24T09:59:27.000Z',
      updated_at: '2022-01-24T10:00:16.000Z',
      deleted_at: null,
      fromCalendar: true,
      disableDrag: false,
    },
  ],
}
```

You can change how the task is rendered with the `itemRenderer` prop
