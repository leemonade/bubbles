## Overview

AvatarsGroup is a component to display a group of avatars in a row. You can control the amount of avatars shown with the `limit` prop

<ThemeProvider>
  <AvatarsGroup
    data={[
      { fullName: 'John Doe' },
      { fullName: 'Mary Jane' },
      { fullName: 'Peter Parker' },
      { fullName: 'Will Teacher' },
      { fullName: 'Tony Stark' },
    ]}
  />
  <Space></Space>
  <AvatarsGroup
    limit={5}
    data={[
      { fullName: 'John Doe' },
      { fullName: 'Mary Jane' },
      { fullName: 'Peter Parker' },
      { fullName: 'Will Teacher' },
      { fullName: 'Tony Stark' },
    ]}
  />
</ThemeProvider>

## AvatarsGroup sizes

AvatarsGroup has four different sizes: `xs`, `sm`, `md` and `lg`.

<ThemeProvider>
  <Group style={{ gap: 48 }}>
    <AvatarsGroup size="xs" data={[{ fullName: 'John Doe' }, { fullName: 'Mary Jane' }]} />
    <AvatarsGroup data={[{ fullName: 'John Doe' }, { fullName: 'Mary Jane' }]} />
    <AvatarsGroup size="md" data={[{ fullName: 'John Doe' }, { fullName: 'Mary Jane' }]} />
    <AvatarsGroup size="lg" data={[{ fullName: 'John Doe' }, { fullName: 'Mary Jane' }]} />
  </Group>
</ThemeProvider>
