## Overview

FileItemDisplay is a component to display a file. Provide a `filename` with an extension, it will automatically separate the name of the file and the extension.

<ThemeProvider>
  <FileItemDisplay filename="Filename.png" />
</ThemeProvider>

## FileItemDisplay options

You can choose to hide the extension or the name with `hideExtension` and `showFileName` respectively

<ThemeProvider>
  <FileItemDisplay filename="Filename.png" hideExtension size={22} />
  <Space style={{ marginBlock: 8 }} />
  <FileItemDisplay filename="Filename.png" showFileName={false} size={22} />
</ThemeProvider>

## FileItemDisplay sizes

You can provide a `size` to change it's size.

<ThemeProvider>
  <FileItemDisplay filename="Filename.png" size={20} />
  <Space style={{ marginBlock: 8 }} />
  <FileItemDisplay filename="Filename.png" size={30} />
</ThemeProvider>

## FileItemDisplay colors

You can provide a `color` to change it's color.

<ThemeProvider>
  <FileItemDisplay filename="Filename.png" size={20} color="#FABADA" />
  <Space style={{ marginBlock: 8 }} />
  <FileItemDisplay filename="Filename.png" size={20} color="blueviolet" />
</ThemeProvider>
