## Overview

Table is a component to display information in a table.

<ThemeProvider>
  <Table
    columns={[
      {
        Header: 'Name',
        accessor: 'name',
      },
      {
        Header: 'Description',
        accessor: 'description',
      },
      {
        Header: 'Actions',
        accessor: 'actions',
      },
    ]}
    data={[
      {
        name: 'Admin',
        description:
          'General managers for the system, has a wide range of permissions except for installing and uninstalling leemons',
        actions: (
          <Stack justifyContent="end" fullWidth>
            <ActionButton tooltip="Open" icon={<ExpandDiagonalIcon />} />
          </Stack>
        ),
      },
      {
        name: 'Manager',
        description:
          'Manages platform users for administrative purposes (logins, deletions, edits and notifications)',
        actions: (
          <Stack justifyContent="end" fullWidth>
            <ActionButton tooltip="Open" icon={<ExpandDiagonalIcon />} />
          </Stack>
        ),
      },
    ]}
  />
</ThemeProvider>

## Table utilization

You must provide an array of `columns` with this shape:

```js
{
  Header: "columnHeader",
  accessor: "columnAccessor"
}
```

And an array of `data`, with the keys of each object in the array matching the column accessors:

```js
[
  { accessor1: 'Column 1, Cell 1', accessor2: 'Column 2, Cell 1' },
  { accessor1: 'Column 1, Cell 2', accessor2: 'Column 2, Cell 2' },
];
```

This example would show this output:

<ThemeProvider>
  <Table
    columns={[
      {
        Header: 'Column 1',
        accessor: 'column1',
      },
      {
        Header: 'Column 2',
        accessor: 'column2',
      },
    ]}
    data={[
      { column1: 'Column 1, Cell 1', column2: 'Column 2, Cell 1' },
      { column1: 'Column 1, Cell 2', column2: 'Column 2, Cell 2' },
    ]}
  />
</ThemeProvider>

If you provide an object to the accessor with a `type = 'chechbox'` property and a `checked` property
it will render a Checkbox cell. You can also provide a node to the accessor to render a custom cell.
