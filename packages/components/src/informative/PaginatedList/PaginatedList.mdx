## Overview

PaginatedList is a component that renders a paginated list. See story for more information.

<ThemeProvider>
  <PaginatedList
    columns={[
      { Header: 'Name', accessor: 'name' },
      { Header: 'Surname', accessor: 'surname' },
    ]}
    items={[
      { name: 'John', surname: 'Doe' },
      { name: 'Manuel', surname: 'Garcia' },
      { name: 'Juan', surname: 'Y medio' },
      { name: 'David', surname: 'Martin' },
      { name: 'Javier', surname: 'Gomez' },
    ]}
    size={5}
    spacing={2}
    totalCount={18}
    totalPages={4}
  />
</ThemeProvider>

## PaginatedList layout

PaginatedList has two different layouts to display a list: `table` and `grid`
