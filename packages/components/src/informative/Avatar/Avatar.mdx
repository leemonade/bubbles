## Overview

Avatar is a component to show a user profile image, initials or a fallback icon

## Avatar initials and icon

If you don't provide an image, you can provide a `fullName` or an `icon`. To display an icon, you must also provide a `fullName` (it shows a random color based on the string) or a `color`.

<ThemeProvider>
  <Group style={{ gap: 16 }} justifyContent="end">
    <Avatar size="md" fullName="Random Name" />
    <Avatar size="md" fullName="Random Name" icon={<StarIcon />} color="blueviolet" />
  </Group>
</ThemeProvider>
