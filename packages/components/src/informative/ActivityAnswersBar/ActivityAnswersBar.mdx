## Overview

ActivityAnswersBar is a component that renders a graphic of the scores of activities.

<ThemeProvider>
  <Box styles={{ height: 300 }}>
    <ActivityAnswersBar
      data={DATA}
      selectables={SELECTABLES}
      labels={{ OK: 'OK', KO: 'KO', null: 'null' }}
    />
  </Box>
</ThemeProvider>

## Bar height

You can change the height of the bars with the `barHeight` prop

<ThemeProvider>
  <Box styles={{ height: 300 }}>
    <ActivityAnswersBar
      barHeight={40}
      data={DATA}
      withSelect={false}
      withLegend={false}
      selectables={SELECTABLES}
      labels={{ OK: 'OK', KO: 'KO', null: 'null' }}
    />
  </Box>
</ThemeProvider>

## Data and selectables

You must provide a `data` array of objects with a `status` property set to `OK`, `KO` or `null`. If you provide an array of selectables,
the object must also have the properties matching the values of the `selectables` array, like this:

```js
{
  id: '6',
  category: 'Mates',
  level: 'Principiante',
  status: 'OK',
},
```

You can also choose to hide the `Select` input.

<ThemeProvider>
  <Box styles={{ height: 300 }}>
    <ActivityAnswersBar
      data={DATA}
      selectables={SELECTABLES}
      withSelect={false}
      labels={{ OK: 'OK', KO: 'KO', null: 'null' }}
    />
  </Box>
</ThemeProvider>
