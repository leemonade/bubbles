## Overview

UserDisplayItem is a component to display an user name with his avatar.

<ThemeProvider>
  <UserDisplayItem
    name={'Ana Maria'}
    surnames={'Lopez Vilchez'}
    rol={'Tutor'}
    email={'bill.sanders@example.com'}
    avatar={
      'https://images.unsplash.com/photo-1508214751196-bcfd4ca60f91?ixid=MXwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHw%3D&ixlib=rb-1.2.1&auto=format&fit=crop&w=255&q=80'
    }
  >
    User display item
  </UserDisplayItem>
</ThemeProvider>

## UserDisplayItem variants and layout

There are four different variants of this component: `inline` (default), `block`, `rol`, `email`.

You can also set the `layout` to the `left` or to the `right`.
