## Overview

ActivityAccordion is a component to render an accordion, you will need to encapsulate the content of the accordion in a `<ActivityAccordionPanel />`, like this:

```jsx
<ActivityAccordionPanel
  label="Test 1"
  color="solid"
  icon={<AlarmBellCheckIcon />}
  rightSection={<Badge label="35" size="md" color="stroke" closable={false} />}
>
  <Box p={20}>
    <Paragraph>Lorem ipsum dolor sit amet ...</Paragraph>
  </Box>
</ActivityAccordionPanel>
```
