## Overview

Calendar is a component that displays a calendar.

<ThemeProvider>
  <Calendar>Calendar</Calendar>
</ThemeProvider>

## Amount of months

With the `amountOfMonths` prop you can change the amount of months that are rendered

<ThemeProvider>
  <Calendar amountOfMonths={4} />
</ThemeProvider>

## Allow level change

You can lock the level (date - month - year) with the `allowLevelChange` prop

<ThemeProvider>
  <Calendar allowLevelChange={false} />
</ThemeProvider>

## Initial level

You can change the initia level that is shown when Calendar is rendered to: `date`, `month` or `year`

<ThemeProvider>
  <Calendar initialLevel="year" />
</ThemeProvider>
