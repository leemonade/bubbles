## Overview

HorizontalStepperContainer is component to render a HorizontalStepper including the steps.

<ThemeProvider>
  <HorizontalStepperContainer
    data={[
      {
        label: 'Resumen',
        badge: 'Badge',
        status: 'OK',
      },
      {
        label: 'Pruebas',
        onClick: () => {},
      },
    ]}
  ></HorizontalStepperContainer>
</ThemeProvider>

## HorizontalStepper width

You can control the width of the HorizontalStepper with the ``navWidth` prop.

<ThemeProvider>
  <HorizontalStepperContainer
    navWidth={400}
    data={[
      {
        label: 'Resumen',
        badge: 'Badge',
        status: 'OK',
      },
      {
        label: 'Pruebas',
        onClick: () => {},
      },
    ]}
  ></HorizontalStepperContainer>
  <HorizontalStepperContainer
    navWidth={800}
    data={[
      {
        label: 'Resumen',
        badge: 'Badge',
        status: 'OK',
      },
      {
        label: 'Pruebas',
        onClick: () => {},
      },
    ]}
  ></HorizontalStepperContainer>
</ThemeProvider>

## HorizontalStepper sticky at

With the `stickyAt` prop, control the position in pixels from the top at which the HorizontalStepper becomes sticky.

<ThemeProvider>
  <HorizontalStepperContainer
    stickyAt={50}
    data={[
      {
        label: 'Resumen',
        badge: 'Badge',
        status: 'OK',
      },
      {
        label: 'Pruebas',
      },
    ]}
  >
