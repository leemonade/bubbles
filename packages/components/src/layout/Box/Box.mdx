## Overview

Box allows you to use the sx prop with any element or component to add inline styles. You can also provide a `padded` prop with a number.

<ThemeProvider>
  <Box
    padding={3}
    sx={(theme) => ({
      backgroundColor: theme.colors.interactive01h,
      cursor: 'pointer',
    })}
  >
    <Text color="primary">Box</Text>
  </Box>
</ThemeProvider>

```jsx
<Box
  padding={3}
  sx={(theme) => ({
    backgroundColor: theme.colors.interactive01h,
    cursor: 'pointer',
    padding: 16,
  })}
>
  <Text>Box</Text>
</Box>
```
