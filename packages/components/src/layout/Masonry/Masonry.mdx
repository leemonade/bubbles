## Overview

Masonry is a component to display an array of elements.

Provide the `breakAt` prop to control the amount of elements for each row, like this:

```js
breakAt: {
  1200: 4,
  940: 3,
  520: 2,
  360: 1,
},
```

Masonry also accepts a `margin` prop to control the gap between elements.

<ThemeProvider>
  <Text style={{ marginBottom: 10, display: 'block' }}>Default margin</Text>
  <Masonry>
    {[0, 1, 2, 3, 4, 5, 6, 7].map((item, index) => (
      <Paper key={index}>
        <Text>{item + 1}</Text>
      </Paper>
    ))}
  </Masonry>
  <Text style={{ marginBottom: 10, display: 'block' }}>30 margin</Text>
  <Masonry margin={30}>
    {[0, 1, 2, 3, 4, 5, 6, 7].map((item, index) => (
      <Paper key={index}>
        <Text>{item + 1}</Text>
      </Paper>
    ))}
  </Masonry>
</ThemeProvider>
