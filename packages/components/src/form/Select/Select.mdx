## Overview

Select is a select input that allows to search, customize dropdown items and value items.

You can allow searching on the select with the `searchable` prop.

<ThemeProvider>
  <Select
    placeholder="Select a value"
    data={[
      {
        label: 'Bender Bending Rodríguez',

## Custom select

You can use your custom dropdown item or value item with `itemComponent` and `valueComponent`. See story for an example.

```jsx
  <Select
    valueComponent={({ label }) => <UserDisplayItem name={label} />}
    itemComponent={({ label }) => <UserDisplayItem name={label} />}
    placeholder="Default variant"
    data={[...]}
  />
```
