## Overview

MultiSelect is an input that allows the selection of multiple values. You can customize the value and list components with: `valueComponent` and `itemComponent`.

## MultiSelect multiple

MultiSelect also accepts a boolean `multiple` prop which is `true` by default. It is used as a `Select` to be able to customize the valueComponent when using a `Select`.

<ThemeProvider>
  <MultiSelect
    placeholder="Multiple to false"
    multiple={false}
    data={[
      {
        image: 'https://img.icons8.com/clouds/256/000000/futurama-bender.png',
