## Overview

ColorPicker is a component that displays an input to choose a color.

<ThemeProvider>
  <ColorPicker />
</ThemeProvider>

## ColorPicker variants

ColorPicker can be `compact` to hide the color swatches and make it smaller:

<ThemeProvider>
  <Group>
    <ColorPicker />
    <Space></Space>
    <ColorPicker compact />
  </Group>
</ThemeProvider>

It can also has a mode for only `useHSL` model:

<ThemeProvider>
  <Group>
    <ColorPicker useHsl />
    <Space></Space>
    <ColorPicker useHsl compact />
  </Group>
</ThemeProvider>

And it can also be set to `fullWidth`:

<ThemeProvider>
  <Box style={{ width: '50%' }}>
    <ColorPicker fullWidth />
  </Box>
  <Box style={{ width: '50%' }}>
    <ColorPicker useHsl fullWidth />
  </Box>
</ThemeProvider>
