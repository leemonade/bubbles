## Overview

Buttons are clickable elements that are used to trigger actions. They
communicate calls to action to the user and allow users to interact with pages
in a variety of ways. `Button` labels express what action will occur when the
user interacts with it.

<ThemeProvider>
  <Button
    color="primary"
    variant="default"
    size="sm"
    leftIcon={<ChevronRightIcon style={{ height: '1.2rem' }} />}
  >
    Button label
  </Button>
</ThemeProvider>

## Button variants

We have three button variants:
**Solid (by default),** outline, and link.

  <Group style={{ gap: 32 }} mt="24px" mb="44px" mx="2px">

    </Button>

      Outline Button
    </Button>
    <Button
      color="primary"
      variant="link"
      size="sm"
      leftIcon={<ExternalLinkIcon style={{ height: '1.2rem' }} />}
    >
      Link Button
    </Button>
  </Group>
</ThemeProvider>

## Rounded style

The rounded style is used for user interface suggested actions for the rest of actions use the default button.

<ThemeProvider>
  <Group style={{ gap: 32 }} mt="24px" mb="44px" mx="2px">
    <Button color="primary" rounded variant="outline" size="sm">
      Skip and go to Users <ChevronRightIcon style={{ height: '1.2rem', margin: '0 -4px 0 8px' }} />
    </Button>
    <Button color="primary" rounded variant="default" size="sm">
      Mach profiles <ChevronRightIcon style={{ height: '1.2rem', margin: '0 -4px 0 8px' }} />
    </Button>
  </Group>
</ThemeProvider>

## Button Colors

We have four button colors:
**Primary (Main actions),** Secondary, Terciary and negative.
Negative color only works for outline and link buttons.

  <Group style={{ gap: 32 }} my="16px" mx="2px">

      Primary Color
    </Button>
    <Button color="secondary" variant="default" size="sm">

    </Button>

### Button `as`

This prop allows you to specify a different element to be rendered as a
`Button`. You may also need to add a [role](#button-role) for accessibility if
an element other than a `button` is rendered.

```jsx
<Button component="div" role="button">
  I'm a div tag
</Button>
```
