## Overview

Textarea is a component that allows the user to input a large text

<ThemeProvider>
  <Textarea showCounter>Textarea</Textarea>
</ThemeProvider>

## Counter

You can show a character count with the `showCounter` prop and with the `counter` you can choose to count between `char` or `word`

<ThemeProvider>
  <Textarea value="This is a Textarea with showCounter and counting characters" showCounter />
  <Box style={{ marginTop: 8 }}></Box>
  <Textarea
    value="This is a Textarea with showCounter and counting words"
    showCounter
    counter="word"
  />
</ThemeProvider>

## Min rows and max lenght

You can also change the amount of minimum rows that are shown and the max lenght of rows it will grow with `minRows` and `maxLenght`
