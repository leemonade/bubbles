## Overview

ProSwitch is a wrapper around the Switch component, so it has the same props, with the addition of `color` and `icon`. That allows you to customize the Switch.

<ThemeProvider>
  <ProSwitch
    size="md"
    color="#FABADA"
    icon={<EmailPileIcon />}
    label="Switch with color and icon changes"
  />
</ThemeProvider>
