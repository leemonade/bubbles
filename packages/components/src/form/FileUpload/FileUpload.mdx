## Overview

FileUpload is a component that allows to upload files.

Drop files here to view how they are displayed in a list. You can remove them by clicking the trash bin icon. You can also
click on the button to open the file browser.

If a file is rejected, an error message will be displayed. You can close the error or click on button to open a file browser.
