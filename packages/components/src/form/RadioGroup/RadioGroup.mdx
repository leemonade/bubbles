## Overview

RadioGroup is a component that groups Radio options.

<ThemeProvider>
  <RadioGroup
    name="overViewRadioGroup"
    data={[
      {
        value: 'option1',
        label: 'Label for option 1',
        icon: <AcademicCapIcon height={32} width={32} />,
      },
      {
        value: 'option2',
        label: 'Label for option 2',
        icon: <AcademicCapIcon height={32} width={32} />,
      },
      {
        value: 'option3',
        label: 'Label for option 3',
        icon: <AcademicCapIcon height={32} width={32} />,
        help: 'Help text for option 3',
        helpPosition: 'bottom',
      },
    ]}
    onChange={(value) => {
      return;
    }}
  />
</ThemeProvider>

## RadioGroup variants
