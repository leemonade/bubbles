## Overview

CheckboxGroup renders a group of `Checkbox`

<ThemeProvider>
  <CheckBoxGroup
    data={[
      { label: 'Option 1' },
      { label: 'Option 2', help: 'Help text', helpPosition: 'bottom' },
      { label: 'Option 3', help: 'Help text', helpPosition: 'right' },
    ]}
  />
</ThemeProvider>

## CheckboxGroup variants

CheckboxGroup has the following variants: `default` and `boxed`.

<ThemeProvider>
  <CheckBoxGroup
    data={[{ label: 'Option 1' }, { label: 'Option 2', help: 'Help text', helpPosition: 'right' }]}
  />
  <Space></Space>
  <CheckBoxGroup
    data={[{ label: 'Option 1' }, { label: 'Option 2', help: 'Help text', helpPosition: 'right' }]}
    variant="boxed"
  />
</ThemeProvider>

## CheckboxGroup orientation

And you can change the orientation of the CheckboxGroup to `column` or `row`.

<ThemeProvider>
  <CheckBoxGroup
    data={[
      { label: 'Row Option 1' },
      { label: 'Row Option 2', help: 'Help text', helpPosition: 'right' },
    ]}
    direction="column"
  />
  <Space></Space>
  <CheckBoxGroup
    data={[
      { label: 'Column Option 1' },
      { label: 'Column Option 2', help: 'Help text', helpPosition: 'right' },
    ]}
    direction="row"
  />
  <Space></Space>
  <CheckBoxGroup
    data={[
      { label: 'Row Option 1' },
      { label: 'Row Option 2', help: 'Help text', helpPosition: 'right' },
    ]}
    variant="boxed"
  />
  <Space></Space>
  <CheckBoxGroup
    data={[
      { label: 'Column Option 1' },
      { label: 'Column Option 2', help: 'Help text', helpPosition: 'right' },
    ]}
    variant="boxed"
    direction="row"
  />
</ThemeProvider>
