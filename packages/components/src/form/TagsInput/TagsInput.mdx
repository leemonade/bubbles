## Overview

TagsInput is a component that lets you select or write tags.

<ThemeProvider>
  <TagsInput labels={{ addButton: 'Add tag' }} />
</ThemeProvider>

## Suggestions

You can provide an array of `suggestions`, to let the user select from a list.

<ThemeProvider>
  <TagsInput labels={{ addButton: 'Add tag' }} suggestions={['Cats', 'Dogs', 'Elephants']} />
</ThemeProvider>
