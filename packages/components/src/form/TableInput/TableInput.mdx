## Overview

TableInput is a component that displays a table where you can input data with different types of inputs. You need to provide a `columns` array with this shape:

```jsx
{
  Header: 'Header',
  accessor: 'accessor'
  input: {
    node: <InputNode />
    rules: {}
  }
  editable: true
}
```

<ThemeProvider>
