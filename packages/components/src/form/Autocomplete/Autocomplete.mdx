## Overview

**Autocomplete** is ...

You must pass a data array to the component, with simple strings or an array of objects with a required **value** property
and **label** property if you are using the multiple Autocomplete.

<ThemeProvider>
  <Autocomplete
    label={'Students'}
    placeholder={'Search for a student'}
    data={[
      { value: 'Ana Alonso', label: 'Ana Alonso', children: 'Ana Alonso' },
      { value: 'Antonia Hidalgo', label: 'Antonia Hidalgo', children: 'Antonia Hidalgo' },
      { value: 'Maria Merino', label: 'Maria Merino', children: 'Maria Merino' },
      { value: 'Juan Carlos', label: 'Juan Carlos', children: 'Juan Carlos' },
    ]}
    nothingFoundLabel={'No maches found'}
  />
</ThemeProvider>

## Custom item and value components

You can display your own custom components in the item list and value.

In this case we will **UserDisplayItem** for the item component and **Badge** for the value component.
