## Overview

We use these buttons for similar task-based actions within a workflow built into the interface so as not to attract too much attention.

## Options

ActionButtons can have only the icon or also include a label with descriptive text to accompany it, through the `label` property. **When we only want the icon, we must also indicate a descriptive text** through the `description` property that will be displayed in a tooltip when hovering over the button:
