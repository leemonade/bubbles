## Overview

ScoreInput is an input that allows the user to select a score.

<ThemeProvider>
  <ScoreInput
    grades={[
      { score: 0 },
      { score: 1 },
      { score: 2 },
      { score: 3 },
      { score: 4 },
      { score: 5 },
      { score: 6 },
      { score: 7 },
      { score: 8 },
      { score: 9 },
      { score: 10 },
    ]}
  />
</ThemeProvider>

## Using letters

If you provide a grade array with letters, something like this:

```js
{
  score: 3.333,
  letter: 'B+',
}
```

And if you pass the `showLetters` prop, it will show the letters instead of the number:

<ThemeProvider>
  <ScoreInput
    showLetters
    grades={[
      {
        score: 0,
        letter: 'F',
      },
      {
        score: 0.667,
        letter: 'D-',
      },
      {
        score: 1,
        letter: 'D',
      },
      {
        score: 1.333,
        letter: 'D+',
      },
      {
        score: 1.667,
        letter: 'C-',
      },
      {
        score: 2,
        letter: 'C',
      },
      {
        score: 2.333,
        letter: 'C+',
      },
      {
        score: 2.667,
        letter: 'B-',
      },
      {
        score: 3,
        letter: 'B',
      },
      {
        score: 3.333,
        letter: 'B+',
      },
      {
        score: 3.667,
        letter: 'A-',
      },
      {
        score: 4,
        letter: 'A',
      },
    ]}
  />
</ThemeProvider>

## Custom score input

You can also input your custom score with the `acceptCustom` prop set to `text` or `number`:

<ThemeProvider>
  <ScoreInput
    acceptCustom="number"
    grades={[
      { score: 0 },
      { score: 1 },
      { score: 2 },
      { score: 3 },
      { score: 4 },
      { score: 5 },
      { score: 6 },
      { score: 7 },
      { score: 8 },
      { score: 9 },
      { score: 10 },
    ]}
  />
</ThemeProvider>

## ScoreInput direction

ScoreInput also allows to change the direction of the grades with the `direction` prop, set to `ltr` or `rtl`

<ThemeProvider>
  <Box style={{ display: 'flex', flexDirection: 'column', gap: 16 }}>
    <ScoreInput
      direction="ltr"
      grades={[
        { score: 0 },
        { score: 1 },
        { score: 2 },
        { score: 3 },
        { score: 4 },
        { score: 5 },
        { score: 6 },
        { score: 7 },
        { score: 8 },
        { score: 9 },
        { score: 10 },
      ]}
    />
    <ScoreInput
      direction="rtl"
      grades={[
        { score: 0 },
        { score: 1 },

        { score: 6 },
        { score: 7 },
        { score: 8 },
        { score: 9 },
        { score: 10 },
      ]}

  </Box>
</ThemeProvider>
