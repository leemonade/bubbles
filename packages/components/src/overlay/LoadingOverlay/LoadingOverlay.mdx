## Overview

LoadingOverlay is a component that renders an overlay over the closest relative element, it takes 100% of the relative element width and height.

You can control if its visible or not with the `visible` prop

<ThemeProvider>
  <Box
    style={{
      height: 100,
      width: 100,
      position: 'relative',
      border: '1px solid black',
      padding: 16,
    }}
  >
    Parent element
    <LoadingOverlay visible />
  </Box>
</ThemeProvider>
