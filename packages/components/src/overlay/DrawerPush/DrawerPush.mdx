## Overview

DrawerPush is an overlay component used to display information from one side of the container.

<ThemeProvider>
  <Box style={{ height: 50, width: '100%', display: 'flex' }}>
    <DrawerPush opened size={280} style={{ padding: 16, border: '1px solid black' }}>
      I'm the content of a DrawerPush
    </DrawerPush>
    <Box style={{ backgroundColor: 'whitesmoke', height: '100%', width: '100%' }}></Box>
  </Box>
</ThemeProvider>

## DrawerPush size

DrawerPush accepts a `size` prop to determine the width or height (depending on the direction) of the DrawerPush

## DrawerPush directions

DrawerPush accepts 4 directions: `ltr` (left to right, `default`), `rtl` (right to left), `ttb` (top to bottom), `btt` (bottom to top).
