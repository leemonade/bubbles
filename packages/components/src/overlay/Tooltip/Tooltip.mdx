## Overview

Tooltip renders a tooltip at a given element on mouse over.

<ThemeProvider>
  <Tooltip label={'Share'} color="primary" style={{ padding: '10px' }}>
    <Button variant="outline">Tooltip primary</Button>
  </Tooltip>
</ThemeProvider>
