## Overview

The Chip component is an element used for displaying small pieces of information or interactive elements. It can represent items such as tags, labels, or clickable buttons with compact visual representations.

<ThemeProvider>
  <Box style={{ padding: 16 }}>
    <Chip> NEW </Chip>
  </Box>
</ThemeProvider>

The Chip can be outlined too

<ThemeProvider>
  <Box style={{ padding: 16 }}>
    <Chip variant="outlined"> NEW </Chip>
  </Box>
</ThemeProvider>

This component had a internal state and a onChange function to control his check state:

<ThemeProvider>
  <Box style={{ padding: 16 }}>
    <Chip variant="outlined" checked={true}>
      {' '}
      NEW{' '}
    </Chip>
  </Box>
</ThemeProvider>
