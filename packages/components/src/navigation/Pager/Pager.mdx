## Overview

Pager is a component to display the current page and navigate between multiple pages

<ThemeProvider>
  <Pager>Pager</Pager>
</ThemeProvider>

## Control the pager

You can control the pager by passing in the following props:

- `total`: The total number of pages.
- `boundaries`: Amount of elements visible on left/right edges.
- `siblings`: Amount of elements visible to the left/right of the selected page.

In this case we have 20 total pages, 2 elements on the edges and the current page has 3 siblings.

<ThemeProvider>
  <Pager
    total={20}
    boundaries={2}
    siblings={3}
    onChange={() => console.log()}
    withControls={false}
  />
</ThemeProvider>

You can also hide/show the controls to move the page and to go directly to the first or last page.

<ThemeProvider>
  <Pager total={10} onChange={() => console.log()} withEdges />
</ThemeProvider>

You can also show a Go To input to write manually the page you want to go.

<ThemeProvider>
  <Pager total={10} withGoTo onChange={() => console.log()} />
</ThemeProvider>

## Pager direction

You can orientate the pager in a horizontal or vertical direction with **direction**

<ThemeProvider>
  <Pager total={3} direction={'column'} withControls={false} onChange={() => void 0} />
</ThemeProvider>

## Disabled state

You can also disable the pager

<ThemeProvider>
  <Pager disabled={true} onChange={() => void 0} />
</ThemeProvider>
