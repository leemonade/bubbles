## Overview

Breadcrumbs is a list of links to display the route to the your current page.

<ThemeProvider>
  <Box style={{ padding: 16 }}>
    <Breadcrumbs
      items={[
        { label: 'Mantine', href: 'https://mantine.dev' },
        { label: 'Mantine hooks', href: 'https://mantine.dev/hooks/getting-started' },
        { label: 'use-id', href: 'https://mantine.dev/hooks/use-id' },
        { label: 'use-id' },
      ]}
    />
  </Box>
</ThemeProvider>

## Back label

You can add a `backLabel` and it will show a button to go to the previous page.

<ThemeProvider>
  <Box style={{ padding: 16 }}>
    <Breadcrumbs
      backLabel="Go back"
      items={[
        { label: 'Mantine', href: 'https://mantine.dev' },
        { label: 'Mantine hooks', href: 'https://mantine.dev/hooks/getting-started' },
        { label: 'use-id', href: 'https://mantine.dev/hooks/use-id' },
        { label: 'use-id' },
      ]}
    />
  </Box>
</ThemeProvider>
