## Overview

HorizontalStepper is a component to display a stepper in an horizontal layout. Control the current step with `currentStep`.
To visualize steps you must provide a `data` array of objects with this shape:

```js
{
  label: 'Step label',
  status: 'Step status' (optional)
}
```

<ThemeProvider>
  <HorizontalStepper data={[{ label: 'Step 1' }, { label: 'Step 2' }, { label: 'Step 3' }]} />
</ThemeProvider>

## Allow step click

HorizontalStepper has a `allowStepClick` prop to allow the user to click a step and navigate the steps.

<ThemeProvider>
  <HorizontalStepper
    allowStepClick
    data={[{ label: 'Step 1' }, { label: 'Step 2' }, { label: 'Step 3' }]}
  />
</ThemeProvider>
