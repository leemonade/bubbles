## Overview

Menu is a component to render a menu button with a list of secondary actions.

<ThemeProvider>
  <Menu
    items={[
      {
        icon: <StarIcon />,
        children: 'Item 1',
        onClick: () => alert('Item 1 clicked'),
      },
      {
        icon: <DeleteBinIcon />,
        children: 'Item 2',
        onClick: () => alert('Item 1 clicked'),
      },
      {
        icon: <FlagIcon />,
        children: 'Item 3',
        onClick: () => alert('Item 3 clicked'),
      },
    ]}
  />
</ThemeProvider>

## Menu arrow and placement

You can also change the menu popover placement to three positions: `center`, `start`, `end`, and choose to display or not an arrow with `withArrow`

<ThemeProvider>
  <Box style={{ display: 'flex', justifyContent: 'space-between' }}>
    <Menu
      placement="start"
      items={[
        {
          icon: <StarIcon />,
          children: 'Item 1',
          onClick: () => alert('Item 1 clicked'),
        },
        {
          icon: <DeleteBinIcon />,
          children: 'Item 2',
          onClick: () => alert('Item 1 clicked'),
        },
      ]}
    />
    <Menu
      withArrow
      items={[
        {
          icon: <StarIcon />,
          children: 'Item 1',
          onClick: () => alert('Item 1 clicked'),
        },
        {
          icon: <DeleteBinIcon />,
          children: 'Item 2',
          onClick: () => alert('Item 1 clicked'),
        },
      ]}
    />
    <Menu
      placement="end"
      items={[
        {
          icon: <StarIcon />,
          children: 'Item 1',
          onClick: () => alert('Item 1 clicked'),
        },
        {
          icon: <DeleteBinIcon />,
          children: 'Item 2',
          onClick: () => alert('Item 1 clicked'),
        },
      ]}
    />
  </Box>
</ThemeProvider>

## Menu gutter

And control the distance between the popover menu and the menu button with `gutter`

<ThemeProvider>
  <Box style={{ display: 'flex', gap: 250 }}>
    <Menu
      position="right"
      gutter={20}
      items={[
        {
          icon: <StarIcon />,
          children: 'Item 1',
          onClick: () => alert('Item 1 clicked'),
        },
        {
          icon: <DeleteBinIcon />,
          children: 'Item 2',
          onClick: () => alert('Item 1 clicked'),
        },
      ]}
    />
    <Menu
      position="right"
      gutter={100}
      items={[
        {
          icon: <StarIcon />,
          children: 'Item 1',
          onClick: () => alert('Item 1 clicked'),
        },
        {
          icon: <DeleteBinIcon />,
          children: 'Item 2',
          onClick: () => alert('Item 1 clicked'),
        },
      ]}
    />
  </Box>
</ThemeProvider>
