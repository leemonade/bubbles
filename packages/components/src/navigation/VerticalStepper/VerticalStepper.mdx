## Overview

VerticalStepper is a component to display a stepper in an vertical layout. Control the current step with `currentStep`.
To visualize steps you must provide a `data` array of objects with this shape:

```js
{
  label: 'Resumen',
  badge: 'Badge' (optional),
  status: 'OK' (optional),
},
```

To add child steps to a step, provide a `childSteps` array with the same shape.

<ThemeProvider>
  <VerticalStepper data={[{ label: 'Step 1' }, { label: 'Step 2' }, { label: 'Step 3' }]} />
</ThemeProvider>

## StepButton

If you add a `onClick` function in any step it will transform into a button step

<ThemeProvider>
  <VerticalStepper
    data={[{ label: 'Step 1' }, { label: 'Step 2', onClick: () => {} }, { label: 'Step 3' }]}
  />
</ThemeProvider>

## Completed and visited steps

You can add a `completedSteps` or `visitedSteps` array to initialize the stepper with completed or visited steps.

<ThemeProvider>
  <VerticalStepper
    completedSteps={[0, 1]}
    visitedSteps={[0, 1]}
    vis
    data={[{ label: 'Step 1' }, { label: 'Step 2', onClick: () => {} }, { label: 'Step 3' }]}
  />
</ThemeProvider>
