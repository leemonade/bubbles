## Overview

Highlight is a component to render text but if you use the property `highlight` you can highlight a part of the text.

<ThemeProvider>
  <Highlight>Text for render</Highlight>
</ThemeProvider>

## Highlight highlight

You can highlight a part of the text with the property `highlight`

<ThemeProvider>
  <Box style={{ display: 'flex', flexDirection: 'column', gap: 16 }}>
    <Highlight>Text rendered without highlighting anything</Highlight>
    <Highlight highlight="part">Text with some part highlighted</Highlight>
    <Highlight highlight="with more text">Other text with more text highlighted</Highlight>
  </Box>
</ThemeProvider>

## Highlight other props

Highlight is a wrapper around Text, so it has all other props of Text.
