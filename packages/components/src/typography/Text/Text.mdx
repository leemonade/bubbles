## Overview

Text is a component to render text.

<ThemeProvider>
  <Text>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. In vel augue sed ante molestie
    pharetra. Aliquam facilisis venenatis iaculis
  </Text>
</ThemeProvider>

## Text size

Text has five different sizes: `xs`, `sm`, `md`, `lg` and `xl`:

<ThemeProvider>
  <Box style={{ display: 'flex', gap: 16 }}>
    <Text size="xs">Extra small text</Text>
    <Text>Small text</Text>
    <Text size="md">Medium text</Text>
    <Text size="lg">Large text</Text>
    <Text size="xl">Extra large text</Text>
  </Box>
</ThemeProvider>

## Text transform

You can transform the text with the `transform` prop, it accepts four different values: `capitalize`, `uppercase`, `lowercase` and `none`.

<ThemeProvider>
  <Box style={{ display: 'flex', gap: 16 }}>
    <Text size="md" transform="capitalize">
      Capitalized text
    </Text>
    <Text size="md" transform="uppercase">
      Uppercase text
    </Text>
    <Text size="md" transform="lowercase">
      Lowercase text
    </Text>
    <Text size="md">Normal text</Text>
  </Box>
</ThemeProvider>

## Text role

You can change the text font to `productive`, `expressive` or `inherit`.

<ThemeProvider>
  <Box style={{ display: 'flex', gap: 16 }}>
    <Text size="md">Expressive text</Text>
    <Text size="md" role="productive">
      Productive text
    </Text>
  </Box>
</ThemeProvider>

## Text colors

## Text bold

You can change the text to bold with `strong`, or if you are using `role="productive"`, you can also use `stronger`

<ThemeProvider>
  <Box style={{ display: 'flex', gap: 16 }}>
    <Text strong>Expressive strong text</Text>
    <Text stronger>Expressive stronger text</Text>
    <Text strong role="productive">
      Productive strong text
    </Text>
    <Text stronger role="productive">
      Productive stronger text
    </Text>
  </Box>
</ThemeProvider>

## Text truncated

If you have a very long text and dont want it to use more than one line, you can use `truncated`

<ThemeProvider>
  <Text truncated>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus ac eros sed mi finibus
    placerat id vitae neque. Vivamus volutpat lacus sed sodales hendrerit. Nullam et elit elementum,
    mattis lectus sit amet, pretium neque. Donec bibendum tempor neque, et malesuada ligula
    tincidunt nec. Proin tincidunt eros non posuere semper. Quisque ac commodo massa. Aenean
    fringilla leo a aliquam ornare. Nam id est lorem. Ut quis urna orci. Pellentesque libero lorem,
    imperdiet eget purus a, vestibulum tristique nibh. Proin felis velit, interdum et quam ac,
    volutpat fermentum risus. Integer nunc erat, sagittis ac lorem et, malesuada mattis leo. Nam
    quis dui in magna feugiat vehicula a et orci.
  </Text>
</ThemeProvider>
