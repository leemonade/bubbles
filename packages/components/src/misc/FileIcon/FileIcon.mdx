## Overview

FileIcon is a component to display a `label` and a icon of the provided `fileType`

<ThemeProvider>
  <FileIcon fileType="audio" label="Audio" />
</ThemeProvider>

## FileIcon types

FileIcon accepts four different fileTypes: `audio`, `video`, `image`, `noicon`.

<ThemeProvider>
  <Box style={{ display: 'flex', gap: 32 }}>
    <FileIcon fileType="audio" label="Audio" />
    <FileIcon fileType="video" label="Video" />
    <FileIcon fileType="image" label="Image" />
    <FileIcon fileType="noicon" label="No icon" />
  </Box>
</ThemeProvider>

## FileIcon color and size

You can change also change the color and size of the label and icon.

<ThemeProvider>
  <Box style={{ display: 'flex', gap: 32 }}>
    <FileIcon fileType="audio" label="Audio" color="#FABADA" size={24} />
    <FileIcon fileType="video" label="Video" color="#015267" size={32} />
    <FileIcon fileType="image" label="Image" color="#ADEBEF" size={64} />
  </Box>
</ThemeProvider>
