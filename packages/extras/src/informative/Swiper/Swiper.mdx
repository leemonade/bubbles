## Overview

Swiper is a component to display a elements in a swiper. You will need the `breakAt` prop
to control how many elements are shown and the space between them, something like this:

```js
breakAt: {
  1200: { slidesPerView: 4, spaceBetween: 20 },
  940: { slidesPerView: 3, spaceBetween: 20 },
  520: { slidesPerView: 2, spaceBetween: 30 },
  360: { slidesPerView: 1, spaceBetween: 40 },
},
```

<ThemeProvider>
  <Swiper
    breakAt={{
      1200: { slidesPerView: 4, spaceBetween: 20 },
      940: { slidesPerView: 3, spaceBetween: 20 },
      520: { slidesPerView: 2, spaceBetween: 30 },
      360: { slidesPerView: 1, spaceBetween: 40 },
    }}
  >

    </Paper>
  </Swiper>
</ThemeProvider>

## Selected and selected styles

Swiper allows you to select the elements with the `selectable` prop, but you can disable the styles of the selected element with `disableSelectedStyles`.

<ThemeProvider>
  <Swiper
    selectable
    breakAt={{
      1200: { slidesPerView: 4, spaceBetween: 20 },
      940: { slidesPerView: 3, spaceBetween: 20 },
      520: { slidesPerView: 2, spaceBetween: 30 },
      360: { slidesPerView: 1, spaceBetween: 40 },
    }}
  >
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
  </Swiper>
  <Box style={{ marginTop: 16 }} />
  <Swiper
    selectable
    disableSelectedStyles
    breakAt={{
      1200: { slidesPerView: 4, spaceBetween: 20 },
      940: { slidesPerView: 3, spaceBetween: 20 },
      520: { slidesPerView: 2, spaceBetween: 30 },
      360: { slidesPerView: 1, spaceBetween: 40 },
    }}
  >
    <Paper fullWidth>
      <Text>Im selectable without styles</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable without styles</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable without styles</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable without styles</Text>
    </Paper>
  </Swiper>
</ThemeProvider>

## Deselectable

With the `deselectable` prop you can control if the element is unselected when you click it again

<ThemeProvider>
  <Swiper
    selectable
    deselectable={false}
    breakAt={{
      1200: { slidesPerView: 4, spaceBetween: 20 },
      940: { slidesPerView: 3, spaceBetween: 20 },
      520: { slidesPerView: 2, spaceBetween: 30 },
      360: { slidesPerView: 1, spaceBetween: 40 },
    }}
  >
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
    <Paper fullWidth>
      <Text>Im selectable</Text>
    </Paper>
  </Swiper>
</ThemeProvider>
