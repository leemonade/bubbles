## Overview

TextEditorInput is a wrapper around TextEditor, it includes all the props of a normal input, for example `label`, `description`, `placeholder` and `error`.

It also includes some tools by default, but you can turn them off with the `toolbars` prop:

```js
toolbars = {
  style: true,
  align: true,
  list: true,
  history: true,
  heading: true,
  color: true,
  formulation: false,
};
```
