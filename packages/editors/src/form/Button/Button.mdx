## Overview

Button are clickable elements that are used to trigger actions.
Buttons need an `icon` and a `label` to give information about the action when hovering.

<ThemeProvider>
  <Button icon={<LadybugIcon />} label="Ladybug" />
</ThemeProvider>

## Actived state

When you set the `actived` prop to true, it will show the actived state.

<ThemeProvider>
  <Button icon={<LadybugIcon />} label="Ladybug" actived />
</ThemeProvider>

## Button positions

Button has different positions: `default`, `start`, `end` and `center`. It will show the `default` position when its alone,
but if it's inside a `ButtonGroup` it will change depending of its position inside the group.

<ThemeProvider>
  <div style={{ display: 'flex', gap: 16 }}>
    <Button icon={<LadybugIcon />} label="Ladybug" position="start" />
    <Button icon={<LadybugIcon />} label="Ladybug" position="center" />
    <Button icon={<LadybugIcon />} label="Ladybug" position="center" />
    <Button icon={<LadybugIcon />} label="Ladybug" position="end" />
  </div>
</ThemeProvider>
